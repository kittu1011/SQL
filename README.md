# LeetCode SQL
Personal solutions *(PostgreSQL)* to LeetCode SQL problems, organized by topic.

## Structure
| Folder | Topic |
|--------|-------|
| `aggregation/` | GROUP BY, HAVING, COUNT, SUM, AVG |
| `ctes/` | Common table expressions, recursive CTEs |
| `date_functions/` | Date arithmetic, EXTRACT, formatting |
| `joins/` | INNER, LEFT, RIGHT, FULL, self-joins |
| `sorting_and_filtering/` | ORDER BY, WHERE, DISTINCT, LIMIT |
| `string_functions/` | LIKE, SUBSTRING, CONCAT, pattern matching |
| `subqueries/` | Correlated subqueries, EXISTS, IN |
| `window_functions/` | ROW_NUMBER, RANK, LAG, LEAD, running totals |

## File naming
```
{problem_number}_{problem_name}.sql
```
Example: `0185_department_top_three_salaries.sql`

## Dialect
All solutions written in **PostgreSQL**.
